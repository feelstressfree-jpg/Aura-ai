# ✨ Aura AI - Next-Gen AI Generation Studio

Aura AI ek powerful aur modern web application hai jo users ko **Text-to-Image** aur **Video Animation** generate karne ki suvidha deta hai—wo bhi bilkul **FREE**. Ise "Flux" aur "Stable Diffusion" jaise industry-leading models ki madad se banaya gaya hai.

## 🚀 Features

- **Instant Image Generation:** Kuch hi seconds mein high-definition (HD) artworks banayein.
- **Video Motion Concepts:** Cinematic motion previews aur animated frames generate karein.
- **100% Free:** No credits, no subscriptions, no login required.
- **Glassmorphic UI:** Ek premium aur professional dark-themed design jo har device par perfect dikhta hai.
- **One-Click Download:** Apni creations ko turant save karein.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Tailwind CSS)
- **Engine:** JavaScript (ES6+)
- **AI API:** Powered by Pollinations.ai (Flux Model)
- **Hosting:** Optimized for Vercel/GitHub Pages
